# ⚽ FOOTBALL TOTAL OVER BOT

Бот мониторит живые матчи и кидает алерты когда игра “горит” —
много ударов в створ, угловых, и матч в нужной минуте.

-----

## 🚀 УСТАНОВКА

### 1. Python

Нужен Python 3.10+. Проверь: `python --version`

### 2. Зависимости

```bash
pip install -r requirements.txt
```

### 3. Создай Telegram бота

1. Напиши @BotFather в Telegram
1. Команда `/newbot`
1. Дай имя и username боту
1. Скопируй токен

### 4. Узнай свой Chat ID

1. Напиши @userinfobot в Telegram
1. Скопируй свой ID

### 5. Вставь данные в bot.py

Открой `bot.py` и замени:

```python
TELEGRAM_TOKEN = "YOUR_BOT_TOKEN_HERE"   # ← твой токен
CHAT_ID = "YOUR_CHAT_ID_HERE"            # ← твой chat id
```

### 6. Запуск

```bash
python bot.py
```

-----

## ⚙️ НАСТРОЙКА ПОРОГОВ

В начале `bot.py` можно менять:

```python
MIN_MINUTE = 58          # С какой минуты ловим сигналы
MAX_MINUTE = 80          # До какой минуты
MIN_SHOTS_ON_TARGET = 5  # Минимум ударов в створ (сумма двух команд)
MIN_CORNERS = 6          # Минимум угловых (сумма двух команд)
CHECK_INTERVAL = 90      # Как часто проверять (в секундах)
```

-----

## 📊 ЛОГИКА СИГНАЛОВ

Бот кидает алерт если ВСЕ условия выполнены:

- ✅ Матч на минуте 58–80
- ✅ Суммарно 5+ ударов в створ
- ✅ Суммарно 6+ угловых
- ✅ Этот сигнал ещё не отправлялся

### Три сценария алертов:

- **Сценарий 1** — счёт 1:0 или 0:1, игра живая
- **Сценарий 2** — счёт 0:0, команды копят давление
- **Сценарий 3** — уже 2+ гола, ещё не конец
